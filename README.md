# Laravel Filament CRM Showcase

Sanitized public mirror of an internal CRM built with Laravel and Filament. It
keeps core application paths and history while removing private runtime files,
assets and generated frontend bundles.

## What This Demonstrates

- Laravel 12 application structure
- Filament resources for companies, contacts, opportunities, tasks and notes
- Policies, models, migrations and seeders for CRM workflows
- Dashboard widgets and admin pages
- Testing and operational runbook structure

## Sanitization Notes

This is not a deployable production dump. Private environment files, generated
assets and internal deployment details were removed.
